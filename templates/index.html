D I O N A

What is our aim?

To address the increased vulnerabilities to cyber attacks and data breaches caused by the rapid increase in IoT devices, Deep Learning methods can be employed. DIONA is a solution that closely observes network traffic between IoT devices and the central control unit using DL methods, providing a two-layered security pattern. It classifies traffic, labels devices, and isolates devices if it detects a threat. This approach is adaptable to all kinds of IoT devices and can be used to ensure cyber security for enterprises and companies.

Project Capabilities

Main Characteristics

DIONA provides Deep Learning integrated Network Security via monitoring applications.

  • Anomaly detection on Network Flow

    DIONA monitors network flow with rules and a DL engine to analyze the traffic status.

  • Anomaly detection on IoT Telemetry

    DIONA monitors IoT telemetries and learn from them. It uses previous knowledge to decide on an behavioral result.

  • Fully clustered

    The better part yet, DIONA is fully dockerized!
